# craft-bunny-storage
A bunny.net storage adapter for CraftCMS

## Requirements
This plugin requires CraftCMS 4.0.0 or later.

## What's it do?
This plugin allows you to use bunny.net Storage as your filesystem for storing assets and the like. It will also allow you to use the bunny CDN (pull zone) to serve said files since you should never be serving them direct from the storage location anyway.
